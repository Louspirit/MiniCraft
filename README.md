Minicraft
=========

Projet du jeu Minicraft.

Requis :
--------
* Java 7
* jMonkeyEngine 3 (build 2013-05-21)

Touches :
---------
* Avancer : Z
* Reculer : S
* Pas lateral a gauche : Q
* Pas lateral a droite : D
* Sauter : espace
* Supprimer un bloc : clic gauche
* Ajouter un bloc : clic droit
* Selection du type de bloc : molette souris
